This project was completed while following a data analytics learning path and focuses on understanding how XLOOKUP improves upon older lookup functions such as VLOOKUP and HLOOKUP.

# Excel Concept Practiced

## XLOOKUP

**XLOOKUP** is a powerful Excel function used to search for values in a dataset and return corresponding results from another column.

Unlike older lookup functions, XLOOKUP allows:

- Looking up values in any direction
- Exact or approximate matches
- Error handling when no match is found
- Simpler and more flexible syntax

Example structure:

XLOOKUP(lookup_value, lookup_array, return_array)

Example use cases in this project:

- Finding employee information using EmployeeID
- Retrieving salary data based on employee names
- Matching values across different columns
